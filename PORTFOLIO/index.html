Insta Clone

I built a full-stack Instagram clone using the MERN stack (MongoDB, Express.js, React.js, Node.js). For image uploads, I integrated Cloudinary, allowing users to securely store and manage media files in the cloud. The application uses a MongoDB Atlas cloud server for scalable database management. The frontend is deployed on Vercel, and the backend is hosted on Render for reliable performance. To enhance user experience with realtime interactions, I implemented Socket.io, enabling instant messaging and real-time notifications.
